* Syntaxe simplifiée
* Permet d'écrire une fonction sur une seule ligne
    ```javascript
    const f = (a, b) => a + b;
    const sum = f(1, 2);
    ```
* Ou non
    ```javascript
    const f = (a, b) => {
      return a + b;
    }
    ```
